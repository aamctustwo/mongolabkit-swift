MongoLabKit is a REST client API for iOS, tvOS and watchOS written to make REST calls to a MongoLab database.

---

## Requirements 

- iOS 9.0+ / Mac OS X 10.9+
- Xcode 7.3 / Swift 2.2

## Platform Support

- iOS
- tvOS
- watchOS
